# Security Policy

## Reporting a Vulnerability

If you discover a security issue, do not open a public issue.

- Email: `carldavenquimoyog@gmail.com`
- Subject: `portfolio-ops security report`
- Include: reproduction steps, impact, affected endpoints/files, and proof-of-concept if available

I will acknowledge the report within 72 hours and provide remediation status updates.

## Scope

Security reports are most helpful when they include issues related to:

- Authentication and session handling
- Authorization and role access control
- Sensitive data exposure
- File upload validation
- Injection and input-validation flaws

## Disclosure

Please allow time for a fix before public disclosure.
