# Contributing

Thanks for your interest in improving `construction-operations-portal`.

## Development Setup

1. Install dependencies:
   - `npm install`
   - `cd backend && npm install && cd ..`
2. Create `.env` from available templates and add required values.
3. Start the app with `npm run dev`.

## Pull Request Guidelines

- Keep changes focused and small.
- Add or update documentation for behavior changes.
- Avoid committing secrets, credentials, or generated artifacts.
- Ensure `npm run build` succeeds before opening a PR.

## Commit Style

Use clear, imperative commit messages, for example:

- `fix auth guard for admin routes`
- `docs update deployment env variables`
- `feat add inquiry status filter`
